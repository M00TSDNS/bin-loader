# bytearray-loader

Byte Array (Uint8Array) loader for webpack

## Installation

```bash
# npm install bytearray-loader
```

## Usage

``` javascript
var fileContent = require("bytearray!./file.bin");
// => returns file.bin content as Uint8Array
```
